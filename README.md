# Password Generator web applications

A secure and user-friendly Password Generator web application built using HTML, CSS, and JavaScript.

## 🔗 Live Demo
[Click here to view the live project](https://anshuraj23.github.io/Password-generator/)

## 📌 Objective
Develop a Password Generator Web Application using HTML, CSS, and JavaScript to demonstrate understanding of basic web development concepts.

## ✨ Features
- Generate random passwords based on user-defined criteria
- Choose password length (4 to 64 characters)
- Select character types:
  - Uppercase letters (A–Z)
  - Lowercase letters (a–z)
  - Numbers (0–9)
  - Special characters (!@#$%^&*)
- Password strength indicator (Weak / Fair / Good / Strong)
- One-click copy to clipboard
- Input validation
- Fully responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla)
